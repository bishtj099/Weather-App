## Description ##
A simple and intuitive weather application built to provide real-time weather updates for any city worldwide. This app fetches live data from a reliable weather API and displays key information such as temperature, humidity, wind speed, and overall conditions in a clean and user-friendly interface.
*Features*
Search weather by city name

Real-time temperature and condition updates

Displays humidity, wind speed, and pressure

Responsive and minimal design for easy use

Built with modern web technologies

 *Tech Stack*
Frontend: HTML, CSS, JavaScript

API: OpenWeatherMap (or whichever API you used)
